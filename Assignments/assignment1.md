Assignment1: Git Account & Branch Creation

1. Create a GitHub account at https://github.com if you do not already have one.
2. Clone your repository to your local machine.
(https://github.com/cyril-arickathil/APP.git)
3. Create a new branch under the main branch using the following commands:

```bash
git checkout main
git pull
git checkout -b <your-branch-name>
```

4. Push the new branch to GitHub:

```bash
git push -u origin <your-branch-name>
```
